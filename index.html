Symantic Framework

Getting Started

Welcome to the Symantic framework written by Bobby Dilley. This framework aims to provide a small and simple way to get started building application. The inspiration came from the Skeleton framework, however I felt that it was a bit lacking in utilities such as forms, buttons and panels. Symantic framework also aims to use all of the new HTML5 semantic elements.
